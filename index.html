This pneumatically actuated leg actuates with the help of elctronically contolled solenoid valves to control the flow of air and has a three linked structure with two joints.A protoptype was done and tested using FRL unit available in the college facilities.
COVID-19 ANALYSIS
This project was done using the help of python modules numpy and pandas using avaliable datasets on a country's happiness factors such as life exptectancy,GDP etc.. and another dataset containing daily covid-19 infection rates for specific countries. The two datasets were filtered and useless columns were dropped and final dataframes were prepared for testing. The correlation between the two dataframes were obtained and finally the realtonship between various life factors and the rate of infection was determined.
AUTONOMOUS ROBOT
This project was done using the help of microcontrollers like arduino UNO ( Bluetooth control ) and/or ESP32 ( WIFI control ) and time of flight sensors. This robot was programmed in such a way that the user can have live camera feedback using the ESP32 module and avoids obstacles and corrects its path such that it doesn't hit the objecct in front of it. The code was written as to detect the obstacles from a pre-defined distance using the TOF sensors.
